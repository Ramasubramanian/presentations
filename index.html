### Your first Java Program ```Java public class FirstClass { public int add(int a, int b) { return a + b; } public static void main(String[] args) { System.out.println(new FirstClass().add(3,4)); } } ``` 

public class FirstClass {
	public int add(int a, int b) {
		return a + b;
	}

	public static void main(String[] args) {
		System.out.println(new FirstClass().add(3,4));
	}
}
## Platform independent - Java code can run on any OS (Mac, Windows, Linux etc) - Java code compiled to bytecode - Bytecode is platform independent and can run on OS specific virtual machine - JVM -> Engineering masterpiece (evolved over 2+ decades) - Bytecode interpreter is OS specific i.e. JVM - .java file -> .class file -> .class file executed by JVM
