## Platform independent - Java code can run on any OS (Mac, Windows, Linux etc) - Java code compiled to bytecode - Bytecode is platform independent and can run on OS specific virtual machine - JVM -> Engineering masterpiece (evolved over 2+ decades) - Bytecode interpreter is OS specific i.e. JVM - .java file -> .class file -> .class file converted to machine code by JVM interpreter

## Memory Management - Garbage Collector (GC) - Automatically frees memory once an Object goes out of scope - Possible to trigger manually but not guaranteed - Global objects are never cleared - Be careful about memory leaks i.e. unintentionally making objects global - Different types of implementations available to choose - Mark and Sweep algorithm - Serial, Parallel, CMS, G1, ZGC etc
## Memory Management ### Stack - To store method local variables and call stacks - Each method call is pushed and popped from call stack once execution is done ### Heap - Used to store global and instance variables - Can set using switches, even in the order of 128+ GBs - Divided into generations based on lifetime of objects
## Memory Management ### Metaspace - Contains class level data or metadata - Separate memory allocation ### Native Memory - Byte buffers

OOPS - Encapsulation

  • Avoid unprotected modifications to data
  • State vs Behaviour

public class SavingsAccount {
  private double balance;
  public void deposit(double amount) {
    balance += amount;
  }

  public void withdraw(double amount) {
    balance -= amount;
  }

  public double getBalance() {
    return balance;
  }
}

OOPS - Inheritance

  • Inheriting state and Behaviour from parent(s)

public abstract class BankAccount {
  protected double balance;
  public abstract void deposit(double amount);
  public abstract void withdraw(double amount);

  public double getBalance() {
    return balance;
  }
}

public class SavingsAccount extends BankAccount {
  @Override
  public void deposit(double amount) {
    balance += amount;
  }

  @Override
  public void withdraw(double amount) {
    balance -= amount;
  }
}

OOPS - Polymorphism

  • Changing behaviour of an Object based on current form
  • Dynamic/Runtime Polymorphism or Method Overriding

public class CurrentAccount extends BankAccount {
  private double transactionFee;

  @Override
  public void deposit(double amount) {
    balance += (amount - transactionFee);
  }

  @Override
  public void withdraw(double amount) {
    balance -= (amount + transactionFee);
  }
}

OOPS - Polymorphism

  • Changing behaviour of an Object based on current form

public interface ZeroBalanceAccount {
  public default boolean canWithdraw(double amount, double balance) {
    return amount - balance >= 0;
  }
}

public class SavingsAccount extends BankAccount {
  private double minimumBalance;
  public void deposit(double amount) {
    balance += amount;
  }

  public void withdraw(double amount) {
    if(this instanceof ZeroBalanceAccount) {
      if((ZeroBalanceAccount(this)).canWithdraw(amount, balance)) {
        balance -= amount;
      } else {
        throw new IllegalStateException("Insufficient funds");
      }
    } else if(amount - balance < minimumBalance) {
      throw new IllegalStateException("Insufficient funds");
    } else {
      balance -= amount;
    }
  }
}

OOPS - Polymorphism

  • Static Polymorphism or Method Overloading or Compile time Polymorphism

public class SavingsAccount extends BankAccount {
  private double minimumBalance;
  public void deposit(double amount) {
    balance += amount;
  }

  @Override 
  public void withdraw(double amount) {
    withdraw(amount, false);
  }

  public void withdraw(double amount, boolean zeroBalance) {
    if(zeroBalance) {
      if(balance >= amount) {
        balance -= amount;
      } else {
        throw new IllegalStateException("Insufficient funds");
      }
    } else {
      if(amount - balance < minimumBalance) {
        throw new IllegalStateException("Insufficient funds");
      } else {
        balance -= amount;
      }
    }
  }
}

public class ZeroBalanceSavingsAccount extends SavingsAccount {
  @Override
  public void withdraw(double amount) {
    withdraw(amount, true);
  }
}
